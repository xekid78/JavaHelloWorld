# JavaHelloWorld
HelloWorldと表示

## 処理
画面にHelloWorldと表示

## コード
```
public class Hello {

	public static void main(String[] args) {
		System.out.println("Hello World");

	}

}
```

## 出力結果  
```
Hello World
```
